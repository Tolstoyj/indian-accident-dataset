# Indian Accident Dataset — Analysis

Analysis and exploratory notebooks for accident counts across **Indian States**, **Union Territories**, and **major Cities** (2021), segmented by gender and age groups.

## Dataset credit

**Source:** [Indian Accident Dataset](https://www.kaggle.com/datasets/adarsh2626/indian-accident-dataset/data) on Kaggle  
**Dataset owner:** **ADARSH KUMAR**  
This project uses the dataset for analysis and visualizations only. All credit for the original data goes to the dataset owner.

## Repository contents

- **Dataset/** — CSV data (accident counts by Type, Region, gender, and age groups).
- **eda_indian_accidents_2021.ipynb** — Exploratory Data Analysis: structure, quality checks, and visualizations (by region, gender, age, heatmaps).

## Data overview

- **Gender:** Male, Female  
- **Age groups:** 0–14, 14–18, 18–30, 30–45, 45–60, 60+ years  
- **Geographic levels:** State, Union Territory, City
